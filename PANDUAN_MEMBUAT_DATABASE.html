ğŸš€ Langkah 1: Install PostgreSQL

Opsi A: Download Installer (Recommended)

  1. Kunjungi website PostgreSQL: https://www.postgresql.org/download/windows/
  2. Klik "Download the installer"
  3. Unduh PostgreSQL 15 atau 16 (stabil dan recommended)
  4. Jalankan installer (postgresql-16.x.exe)
  5. Instalasi dengan konfigurasi:

  7. PostgreSQL akan auto-start setelah instalasi selesai

ğŸ†˜ Troubleshooting

Error 1: "psql is not recognized"

Penyebab: PostgreSQL belum ditambahkan ke PATH Windows

Solusi:

  1. Tekan Win + R
  2. Ketik environment dan tekan Enter
  3. Klik "Edit the system environment variables"
  4. Klik "Environment Variables" button
  5. Di "System variables", cari Path dan klik "Edit"
  6. Klik "New" dan tambahkan: C:\Program Files\PostgreSQL\16\bin
  7. Klik OK 3x
  8. Restart PowerShell dan coba lagi: psql --version

Error 2: "Connection refused at 127.0.0.1:5432"

Penyebab: PostgreSQL server tidak berjalan

Solusi:

  1. Tekan Win + R â†’ ketik services.msc â†’ Enter
  2. Cari "postgresql-x64-16"
  3. Klik kanan â†’ Start

Error 3: "Database 'primary_school' already exists"

Penyebab: Database sudah pernah dibuat sebelumnya

Solusi:

npm run prisma:migrate reset

Pilih "y" saat ditanya. Ini akan menghapus dan recreate database.

Error 4: ".env.local not found" atau "DATABASE_URL not defined"

Penyebab: File .env.local tidak ada atau belum disimpan

Solusi:

  1. Pastikan file ada di: c:\Users\DUNIA IMAJINASIKU\Documents\Web\primary-school\.env.local
  2. Tutup terminal dan buka terminal baru
  3. Terminal lama tidak akan membaca .env.local yang baru

Error 5: "Port 3000 already in use"

Penyebab: Aplikasi Next.js sudah berjalan di port 3000

Solusi:

$env:PORT=3001
npm run dev

Buka http://localhost:3001 di browser.
